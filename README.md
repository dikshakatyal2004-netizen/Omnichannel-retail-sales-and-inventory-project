# 📊 Omnichannel Retail Sales and Inventory Project

## 📌 Project Overview
This project analyzes retail sales data to understand business performance, customer behavior, and product trends using Excel, SQL, and Power BI.

## 🛠 Tools & Technologies
- Excel (Data Cleaning & Preprocessing)
- SQL (Data Analysis & Queries)
- Power BI (Dashboard Visualization)
- Git & GitHub (Version Control)

---



---

## 📈 Key Insights
- Sales performance varies significantly across regions.
- Peak sales observed during high-demand months (seasonality trend).
- Certain product categories contribute the highest revenue.
- Customer segmentation shows majority revenue from a specific segment.

---

## 💡 Business Recommendations
- Focus marketing efforts on high-performing regions.
- Increase inventory for top-selling products.
- Improve strategies in low-performing regions.
- Implement retention strategies for repeat customers.

---

## 📁 Dataset
- Retail sales dataset containing product, region, and customer details.
- Data cleaned and transformed using Excel.

---

## 🚀 Project Outcomes
- Built an interactive dashboard using Power BI
- Performed data cleaning and preprocessing
- Generated actionable business insights
- Wrote SQL queries for data analysis
